## 1. 动机：为什么需要 Macro MIF

### 1.1 微观 MIF 的天然边界

微观 MIF 在 DOM 层面工作良好：20 层×2 侧的快照就是完整的场状态 Ψ，所有算子（H_m、I_em、Ω、div J）都有精确定义。但它有一个结构性盲区——**它不知道 DOM 外面的世界在发生什么**。

一个 BTC DOM 可以在某一分钟内完美呈现"高 Ω、结构清晰、ICI 强信号"的状态，但如果同一时刻 Fed 正在做紧急回购操作、储备急剧下降、整个固定收益市场的流动性在收缩——这些信息不会出现在任何一个交易所的 DOM 里，却会在接下来几小时到几天内决定所有风险资产的走向。

微观 MIF 看到的是"房间里的天气"，Macro MIF 试图看到"窗外的气候"。

### 1.2 核心公理

**资产价格是资金流通过流动性结构的投影。**

宏观层面的"order flow"就是资金流——Fed 资产负债表、TGA、RRP 之间的资金搬运。这些流动不产生 DOM 上的直接痕迹，但它们决定了所有 DOM 存在的"环境温度"。

## 2. 哲学基础：与微观 MIF 的统一性

### 2.1 当前状态：结构相似，尚非哲学统一

微观 MIF 的 Ψ 是直接测量的——DOM 快照就是场本身。宏观 MIF 的"场变量"（储备、SOFR 利差、GEX 等）不是场本身，是**场的间接投影**。这个区别类似于天文学中"通过光谱推断恒星成分"——合理，但你没有直接触碰恒星。

两者一致的部分：都把"结构而非价格"作为研究对象；都有 regime 分类（IS/UIS 对应 Normal/Stress）；都承认认知边界；都用证伪逻辑。

不一致的部分：微观 Ω 有严格的数学定义（三分量几何平均），宏观没有对应的单一序参量；微观的测不准原理（ΔE×Δt≥ℏ_m）在宏观没有对应物；微观的事件元（LP/RP/DP/OP）在宏观没有等价定义。

### 2.2 统一路径：通过大数定理构造 Ψ_macro

**核心论证**：微观 Ψ 是直接测量的，宏观 Ψ_macro 可以通过大数定理（CLT）间接获取。

思路类似统计力学从 10^23 个分子推导出 PV=nRT。金融系统中，SOFR 不是一个人的借贷利率，它是几千亿美元隔夜回购市场上所有参与者行为的聚合统计量。SOFR 的每日读数本身就是大数定理的产物——大量独立（或弱相关）的借贷决策聚合后，涌现出一个稳定的"价格信号"。当这个信号偏离 IORB，偏离本身就是一个低噪音的充分统计量，告诉你储备分布的某个关键特征。

因此 Ψ_macro 不需要是"所有资金管道的完整描述"。它只需要是**一组充分统计量**——在统计意义上，这组变量包含了你在给定观测条件下能提取的全部信息。

### 2.3 Ψ_macro 的正式定义

```
Ψ_macro(t) = {
  stock:  [WRESBAL, WALCL, TGA, RRP]      — 会计恒等式约束的存量状态
  price:  [SOFR-IORB, FRA-OIS]            — CLT 聚合的流量压力信号
  risk:   [RV/IV, IC, GEX_sign]           — CLT 聚合的风险定价信号
}
```

**每一层的信息论理由**：

- **stock 层**：被会计恒等式锁死。ΔReserves = ΔWALCL − ΔTGA − ΔRRP。零自由度。这不是模型，是记账规则。相当于牛顿力学的第一定律——不管中间发生了什么复杂的事情，最终储备的变化只能来自这三个源。

- **price 层**：市场聚合的充分统计量。SOFR-IORB 利差是储备分布均匀性的最灵敏指标——所有储备充裕性的问题最终都会在这个利差上留下痕迹。就像微观 MIF 中 best bid/ask 的 ρ 是最高信息密度的位置。FRA-OIS 3M 是远期流动性预期的聚合。

- **risk 层**：衍生品市场聚合的充分统计量。RV/IV 告诉你实际波动率与市场预期的偏差（已定价 vs 未定价风险），Implied Correlation 告诉你分散化假设是否成立，GEX 符号告诉你做市商的对冲方向。

### 2.4 Ψ_macro 上的算子

微观 MIF 已有的全部算子可以在 Ψ_macro 上自然定义：

**H_macro**：Ψ_macro 各分量的联合熵。当所有分量高度相关（stress 时趋于同步），H 下降，"可读性"上升。

**Ω_macro**：类比微观的三分量几何平均。ω_s_macro（stock 稳定性）× ω_p_macro（price 信号一致性）× ω_t_macro（时间持续性）。

**div J_macro**：Ψ_macro 状态的变化方向。储备在降但利差没动 → 信息还没传到价格 → div J < 0 → 凝聚中。储备在降且利差在扩 → 信息已经传导 → div J ≈ 0 → 已定价。

这样做之后，微观和宏观不再只是"结构相似"，而是**同一套算子在不同 configuration space 上的实例化**。

### 2.5 信息清晰度条件

观察者什么时候"看得更清楚"？当多个独立管道同时发出一致信号的时候。

SOFR 利差扩大 + 储备下降 + RRP 缩量——三件事指向同一个方向（流动性在收缩），信息熵急剧下降。这就是宏观的"IS 态"。反过，SOFR 平稳但储备在动、RRP 也在动但方向不一致——信息熵高，你看不清，这是"UIS 态"。

### 2.6 物理类比

- **守恒律**：ΔReserves = ΔWALCL − ΔTGA − ΔRRP（会计恒等式 = 牛顿第一定律）
- **时间尺度分离**：L0（季度）/ L1（周）/ L2（日）= 热力学宏观/微观分离
- **坐标系**：(stock, price, risk) 三轴在正常 regime 下近似正交——储备可以充裕但风险突然重定价（SVB），也可以储备紧张但风险无人在意（2018 年底 QT）

## 3. 三层架构

### 3.1 L0：边界层（季度级）

定义系统运行的"宪法约束"。变化极慢，一旦设定就锁定几个月的动力学空间。

| 变量 | 频率 | 来源 | 作用 |
|------|------|------|------|
| Fed Funds Rate（目标区间） | FOMC 会议 | Fed | 资金成本的绝对锚 |
| QT 速度（月度到期不续） | FOMC 指引 | Fed | 储备消耗速度上限 |
| QRA 发债计划 | 季度 | 财政部 | TGA → 储备的搬运计划 |
| SLOOS（银行贷款意愿） | 季度 | Fed | 信贷传导通道的阻力 |

L0 变量几乎不需要实时监控。它们是"已知的已知"，只在 FOMC 会议或 QRA 发布时更新。

### 3.2 L1：场层（日频/周频）

两个场共同描述系统的实时状态。

#### Φ_liquidity（流动性场）

核心会计恒等式：
```
ΔReserves_implied = ΔWALCL − ΔWTREGEN − ΔRRPONTSYD
```

| 指标 | FRED 代码 | 频率 | 角色 |
|------|-----------|------|------|
| Fed 资产（SOMA 持有量） | WALCL | 周三 | QT 的实际执行速度 |
| TGA（财政部一般账户） | WTREGEN | 周三 | 财政收支对储备的冲击 |
| ON RRP 余额 | RRPONTSYD | 日频 | "缓冲垫"的厚度 |
| 银行储备 | WRESBAL | 周三 | 结果变量（被前三者锁定） |
| SOFR | Fed/FRED | 日频 | 管道最窄处的压力读数 |
| IORB | Fed | FOMC | 储备的机会成本锚 |
| FRA-OIS 3M | Bloomberg/FRED | 日频 | 远期流动性预期 |
| SOFR 99th %ile | Fed | 日频 | 分布尾部——储备分布不均匀度 |

**SOFR-IORB 利差的特殊地位**：这是宏观 MIF 中信息密度最高的"窄管道"，类比微观 MIF 中 best bid/ask 的 ρ。所有储备充裕性的问题最终都会在这个利差上留下痕迹。Sep 2019 回购危机的教训正是：SOFR 99th %ile 是事后被识别出来的"缺失维度"——表面上储备"充裕"，但分布极不均匀，这个不均匀性在 SOFR 99th %ile 上早有预兆。

#### Φ_risk（风险场）

| 指标 | 来源 | 频率 | 角色 |
|------|------|------|------|
| GEX 符号 + 翻转水平 | SpotGamma/自行估计 | 日频 | 做市商对冲方向 |
| Implied Correlation（ICJ） | CBOE/FRED: CICJ3M | 日频 | 分散化假设是否成立 |
| IC 期限结构 1M vs 6M | 衍生 | 日频 | 短期恐慌 vs 结构性担忧 |
| CoT 净投机持仓 Δ（z-score） | CFTC.gov | 周频 | 机构持仓的边际变化方向 |

### 3.3 L2：动力学层（滚动日频）

在 L1 的场变量上计算动态特征，进行 regime 分类。

#### 实现波动率 vs 隐含波动率比（RV/IV）

```
RV_21 = σ(daily log returns, 21d) × √252 × 100
Ratio = RV_21 / VIX
```

Ratio < 0.8：已实现波动率低于市场预期 → 市场在"卖保险"状态，risk-on。
Ratio > 1.2：已实现波动率超过预期 → 市场被 surprise，风险未充分定价。

#### 方差比 VR(q)

```
VR(q) = Var(q-day returns) / [q × Var(1-day returns)]
q = 5, 10, 21，在 SPX、TY、DXY、Gold 上分别计算
```

VR(q) = 1：纯随机游走。
VR(q) > 1：趋势（正自相关），momentum 策略有效。
VR(q) < 1：均值回归（负自相关），contrarian 策略有效。

#### 联合压力判断

```
0 个场处于 stress → Normal
1 个场处于 stress → Monitor
2+ 个场同时 stress → Structural stress（多尺度退相干，类似微观 MSI 退相干退出信号）
```

## 4. 隐变量处理：影子银行与实际利率

### 4.1 问题本质

影子银行规模和实际利率有一个共同特征：它们是 **latent variables**——永远观测不到真值，只能通过多个 noisy proxy 间接推断。

影子银行：Fed 的 Z.1 Financial Accounts 给季度估计，FSB 给年度估计，BIS 给另一组数字。这些估计之间可以差万亿美元级别。你拿到的不是一个数，是一个**分布**。

实际利率：名义利率你知道，但"通胀预期"这个减数本身就是估计。TIPS breakeven 是一个 proxy，Cleveland Fed 的模型是另一个 proxy，Survey of Professional Forecasters 是第三个。它们经常分歧。

### 4.2 设计原则：隐变量作为不确定性修正项，不作为状态维度

隐变量**不进入** Ψ_macro 的状态空间本身。它们进入的是 Ω_macro 的**折损因子**。

逻辑是：当你对这些 latent variables 的不确定性高时，你对整个系统状态的可读性应该打折。你不假装知道影子银行的精确规模，你只承认"当我对它的估计越不确定，我对整体局势的判断就越不确定"。

### 4.3 η 折损因子

对每个 latent variable，计算其 proxy 之间的分歧度：

```
σ_shadow = std(FSB_est, Z.1_est, BIS_est) / mean(...)
σ_r_star = std(TIPS_breakeven, Cleveland_model, SPF) / mean(...)
```

这个变异系数就是你对该 latent variable 的"认知模糊度"。

信息折损因子：
```
η = exp(-λ₁ × σ_shadow - λ₂ × σ_r_star)
```

η ∈ (0, 1]。所有 proxy 高度一致（σ 小），η ≈ 1，Ω_macro 不受影响。proxy 严重分歧（σ 大），η 下降，Ω_macro 被折损——你承认"我看不清这部分"。

最终：
```
Ω_macro_adjusted = Ω_macro × η
```

### 4.4 隐变量交叉相关性追踪

```
C_latent(t) = rolling_corr_matrix([σ_shadow, σ_r_star, ...], window=60d)
```

当 C_latent 的最大特征值突然上升，说明 latent variables 开始同步运动——某个你没直接观测到的系统性力量在起作用。这和微观 MIF 中 I_em（买卖互信息）上升的逻辑完全类似：独立分量突然耦合 = 信息结构在变化 = 要么是信号，要么是风险。

### 4.5 诚实的边界

定义漂移（如"影子银行" = 2006 年的 SIV vs 2024 年的私人信贷基金）**不可用 CLT 解决**。CLT 假设你测的是同一个 θ，只是有噪音。如果 θ 本身在漂移，你的所有 proxy 都会同时失灵，而且你不知道它们失灵了——因为它们可能仍然高度一致（一致地测错了东西）。这是范式危机场景，需要定义审计层处理。

## 5. Regime 适应：季度审计与遗忘系数 γ

### 5.1 两层分离

将"参数校准"（自动、高频、窄范围）和"定义审计"（人工、季度、可能破坏性）严格分开。

**第一层：参数校准（自动）**

对象：分位数阈值、λ 权重、相关性窗口长度。

方法：每月用最近 6 个月的数据重新计算分位数基准。不需要人工介入，无过拟合风险（分位数是非参数的）。这是 ADR-002（分位数阈值）的宏观版本。

**第二层：定义审计（人工，季度触发）**

触发条件：Fed 或 BIS 发布以下任何一类文件——Financial Stability Report、Z.1 Flow of Funds 修订、FSB Global Monitoring Report、或 Fed 关于 non-bank 的新监管框架。

三个审计问题：
1. 当前 proxy 列表是否还覆盖了报告里标记为"systemic"的实体类别？
2. 现有 proxy 的定义口径是否发生了变化？
3. 报告里是否出现了新的风险传导路径，使得之前假设"弱相关"的 latent variables 变成了强相关？

如果三个问题的答案都是"没变化"，什么都不做。

### 5.2 变更规则（不对称摩擦）

- **加维度**：立即执行。成本低（只是多一个输入），风险低（如果没用，η 计算中它的权重自然会很小）。
- **减维度**：标记为"degraded"并降低权重。只有连续两个季度持续 degraded 才真正移除。这个滞后是故意的——防止因为一次数据中断就改变状态空间。
- **改 η 的 λ 权重**：这是最危险的操作。λ 只能在年度 review 时调整，且调整幅度不超过 ±20%，且必须有样本外证据支持（过去 12 个月里，调整后的 Ω_macro 在已知 stress 事件中的信号质量是否显著优于调整前）。

### 5.3 遗忘系数 γ（范式重置惩罚）

每次范式重置（参数重定义）后，系统应该承认"我刚换了一副眼镜，还没验证这副眼镜是否准确"，然后随着新证据积累逐步恢复信心。

```
γ(t) = 1 - (1 - γ₀) × exp(-t/τ_recovery)

γ₀ = 0.5     （重置时刻的初始折损：对新参数只有一半信心）
τ_recovery = 90 days（恢复时间常数：大约一个季度的新数据）
```

最终可读性：
```
Ω_macro_final = Ω_macro × η × γ(t)
```

η 处理 latent variable 的测量不确定性，γ 处理参数本身的"新鲜度不确定性"。两个乘法因子，各管各的。

**连续重置（记忆性）**：
```
γ_new = γ_current × γ₀  （而非重置到 γ₀——历史累积）
```

如果半年内连续三次重置，γ ≈ 0.5³ = 0.125，系统进入"暂停判断"状态。这和微观框架里的范式危机处理（停止交易 30 天）是同一个精神，只是宏观版本用连续函数替代了二值开关。

这个设计自然惩罚了频繁调参。如果你每个月都重置一次，γ 永远在 0.5-0.7 之间徘徊，Ω_macro 被持续压低，系统几乎不会给出高置信信号。这迫使你只在真正必要时才触发重置——因为每次重置的代价是几个月的"视力模糊期"。过拟合的最好防御不是禁止调参，而是让调参的代价足够高。

**关于连续三次重置的概率**：Fed 作为一个被全球资本市场持续博弈和纠错的机构，其发布的定义和数据口径的稳定性，本身就是大量参与者长期压力测试的均衡结果。对 Fed 专业性的信任本身也是大数定理——这个纠错机制的可靠性由参与者数量保证。因此 γ 的连续折损逻辑保留作为理论完备性的边界条件即可，不需要在实现中特别关注。实际运行中 γ₀ = 0.5、τ = 90 天、一年最多触发一两次，这就是它的正常工作区间。

### 5.4 采样频率不一致的处理

Ψ_macro 各分量的采样频率不一致。stock 层是周频，price 层是日频，risk 层有日频也有周频（CoT）。在算 H_macro 或 Ω_macro 时，快变量和慢变量混在同一个状态向量里，慢变量在两次更新之间是"假静止"的——不是没变，是你不知道它变了多少。

微观 MIF 没有这个问题，20 层 DOM 是同一个时刻的快照，所有分量同步。

处理方式：给每个分量加一个**信息新鲜度衰减**，和 γ 的思路完全一致：

```
freshness_i(t) = exp(-(t - t_last_update_i) / τ_i)
```

τ_i 和该分量的典型变化速率挂钩：
- SOFR：τ = 1 天（日频更新，衰减慢）
- WRESBAL：τ = 2-3 天（周频更新，中间几天信息在老化）
- CoT：τ = 3-4 天（周频且报告本身就滞后）

然后 Ω_macro 的计算中，每个分量的贡献按 freshness 加权。效果是：周四刚发布储备数据后 Ω_macro 最可靠，到下周三接近下次发布前 Ω_macro 自动被压低——不是因为市场变了，是因为你承认自己的信息在变旧。

## 6. 局限与担忧

### 6.1 CLT 收敛在 stress 时退化

大数定理保证 proxy 误差以 1/√N 速度收缩，但前提是误差之间弱相关。在危机时，所有人的模型同时失灵（proxy 误差变成强相关），CLT 的收敛速度在你最需要它的时候反而最差。

η 折损因子部分缓解了这个问题（proxy 分歧度上升 → η 下降 → Ω_macro 自动降低），但这是"事后发现分歧"的机制。如果所有 proxy 一致地错了，η 不会响应。这就是 Section 4.5 中描述的范式危机场景。

### 6.2 定义漂移不可用 CLT 解决

"影子银行"在 2006 年是 SIV 和 CDO，在 2024 年是私人信贷基金和加密稳定币。proxy 可以一致地测对旧定义，同时完全遗漏新定义下的风险。这不是测量误差问题，是测量对象变了。

唯一的防御是 Section 5.1 中的季度定义审计。但审计是人工的、滞后的、不完美的。这个边界无法通过算法消除。

### 6.3 Ψ_macro 的"充分性"假设

整个框架建立在一个可证伪的假设上：Ψ_macro 的这组变量是"充分的"——即在给定观测条件下，它们包含了你能提取的全部信息。如果存在你没观测到的重要状态变量，Ω_macro 会给出虚假的"高可读性"信号。Sep 2019 回购危机就是一个例子——表面上储备"充裕"，但分布极不均匀，这个不均匀性在当时的常规观测变量里几乎不可见。

SOFR 99th %ile 被纳入正是因为这个教训。但未来的"缺失维度"无法提前预知。

### 6.4 宏观场变量的因果模糊性

微观 MIF 中，DOM 变化 → 价格变化的因果链是直接的（大量吃单 → 价格移动）。宏观层面，储备变化 → 资产价格变化的因果链经过多层中介（储备 → 银行间借贷成本 → 信贷供给 → 资产估值），延迟长且非线性。Ω_macro 高并不意味着你能精确预测"什么时候"和"什么资产"会动。

## 7. 核心公式汇总

### 7.1 会计恒等式（守恒律）

```
ΔReserves = ΔWALCL − ΔTGA − ΔRRP
```

不是模型，是记账规则。零自由度。

### 7.2 Ψ_macro 定义

```
Ψ_macro(t) = {
  stock:  [WRESBAL, WALCL, TGA, RRP]
  price:  [SOFR-IORB, FRA-OIS_3M]
  risk:   [RV_21/VIX, IC_3M, GEX_sign]
}
```

### 7.3 RV/IV 比

```
RV_21 = std(daily_log_returns, window=21) × √252 × 100
Ratio = RV_21 / VIX
```

### 7.4 方差比

```
VR(q) = Var(q-day returns) / [q × Var(1-day returns)]
q ∈ {5, 10, 21}，对 SPX, TY, DXY, Gold 分别计算
```

### 7.5 η 折损因子

```
σ_shadow = std(FSB_est, Z.1_est, BIS_est) / mean(...)
σ_r_star = std(TIPS_breakeven, Cleveland_model, SPF) / mean(...)
η = exp(-λ₁ × σ_shadow - λ₂ × σ_r_star)
Ω_macro_adjusted = Ω_macro × η
```

### 7.6 γ 遗忘系数

```
γ(t) = 1 - (1 - γ₀) × exp(-t/τ_recovery)
γ₀ = 0.5, τ_recovery = 90 days
连续重置：γ_new = γ_current × γ₀
Ω_macro_final = Ω_macro × η × γ(t)
```

### 7.7 信息新鲜度衰减

```
freshness_i(t) = exp(-(t - t_last_update_i) / τ_i)
τ_SOFR = 1d, τ_WRESBAL = 2-3d, τ_CoT = 3-4d
```

### 7.8 隐变量交叉相关性

```
C_latent(t) = rolling_corr_matrix([σ_shadow, σ_r_star, ...], window=60d)
最大特征值 spike → 隐变量同步 → 未观测的系统性力量活跃
```

## 8. 验证计划（待执行）

### 8.1 回测证伪测试

**Baseline**：仅用 VIX + 储备水平的简单规则。

**Full model**：L0/L1/L2 联合 regime 判断。

**测试窗口**：
- Sep 2019 回购危机（SOFR spike，储备分布不均）
- Mar 2020 COVID 冲击（外源 shock，所有场同时 stress）
- Mar 2023 SVB（储备充裕但风险突然重定价）

**成功标准**：Full model 在至少 2/3 窗口中信号质量显著优于 baseline。

**证伪条件**：Full model 不优于 baseline → 宏观 MIF 的多层架构没有增量信息。

### 8.2 Φ_liquidity 与 Φ_risk 的条件独立性

量化两个场在 Normal regime 下的条件独立性，以及在 Stress regime 下独立性崩溃的程度。如果两个场在所有 regime 下高度相关，那分开建模没有增量价值。

### 8.3 η 参数校准

用已知事件标定 λ₁、λ₂ 的合理范围。在 proxy 分歧度高的时期，Ω_macro_adjusted 是否确实比 Ω_macro 给出更保守（更准确）的可读性判断。

### 8.4 γ 和 τ_recovery 的经验标定

在历史上的参数重定义事件（如 2013 年 taper tantrum 后 Fed 修改 forward guidance 框架、2019 年回购危机后 Fed 创建 SRF）后，系统需要多长时间才能重新给出可靠信号。

## 9. 与微观 MIF 的映射关系

| 维度 | 微观 MIF | 宏观 MIF |
|------|---------|---------|
| Configuration | Ψ = {ε(p,s,t)} DOM 快照 | Ψ_macro = {stock, price, risk} |
| 测量方式 | 直接 | 间接（CLT 聚合 + 会计恒等式） |
| 序参量 | Ω = [ω_s×ω_t×ω_p]^(1/3) | Ω_macro = [ω_s_macro×ω_p_macro×ω_t_macro]^(1/3) |
| 信息流 | div J = ∂H_m/∂t - ∂I_em/∂t | div J_macro = stock→price 传导状态 |
| 不确定性来源 | 测不准原理 ΔE×Δt≥ℏ_m | η 折损 + γ 遗忘 + freshness 衰减 |
| IS/UIS 判断 | Ω vs 分位数阈值 | Ω_macro_final vs 分位数阈值 |
| 守恒律 | dI/dt + div(J_I) = S_ext | ΔReserves = ΔWALCL − ΔTGA − ΔRRP |
| 最高信息密度位置 | best bid/ask 的 ρ | SOFR-IORB 利差 |
| 防守弹性 | κ (DOM autocorr) | η (proxy 一致性) |
| 范式危机 | 停止交易 30 天 | γ 连续折损至暂停判断 |
