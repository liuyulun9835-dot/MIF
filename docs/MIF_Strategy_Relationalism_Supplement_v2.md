## 1. 策略信条体系

### 1.1 六大信条

| 信条 | 核心 | 实现方式 | 违反风险 |
|------|------|---------|---------|
| 1. 关系优于点 | 分位数而非绝对值 | quantile(x, p) | 🔴 高 |
| 2. 动态优于静态 | 导数而非瞬时值 | dΩ/dt, d²Ω/dt² | 🔴 高 |
| 3. 矢量优于标量 | 多维而非单值 | E=(dir,mag,qual) | 🟡 中 |
| 4. 分布优于期望 | 概率而非二值 | confidence∈[0,1] | 🟡 中 |
| 5. 窗口优于瞬时 | 统计量而非单点 | std_M, autocorr_W | 🟢 低 |
| 6. 相位优于幅度 | 内积而非范数和 | R=z(I)×z(E) | 🟢 低 |

### 1.2 v2.4 新增的隐含信条

**信条 7（新）：双重确认优于单一度量**
```
来源：v2.4 Hawkes 判别协议
核心：IS 确认需要结构层 I(Γ) 和事件层 n 同时通过
      单一度量的信号被视为噪声

实现：
- 不用 I(Γ) > I_c 即入场（单一确认）
- 用 I(Γ) > I_c AND n ≥ n_threshold（双重确认）

物理意义：
结构有序（高 Ω）不等于突破真实
突破真实（高 n）不等于结构稳定
两者缺一不可
```

**信条 8（新）：动能衰减优于价格反转**
```
来源：v2.4 Hawkes 离场机制
核心：离场信号是动能耗尽（λ→μ），不是价格反转

实现：
- 不用"价格跌破 X 止损"（空间锚点）
- 用"λ_excite → μ 且 ∂²λ/∂t² < 0"（动能衰减）

物理意义：
动能耗尽意味着系统回到鞅态
此时 alpha 已经消失，即使价格还没反转
等到价格反转再走 = 将已赚利润暴露于零期望的随机游走中
```

## 3. 微分方程解视角

### 3.1 高阶导数的信息价值

```
零阶（位置）：Ω_t
  只知道"现在可读性多高"
  解空间：无穷多条轨迹

一阶（速度）：dΩ/dt
  知道"可读性在上升还是下降"
  解空间大幅收窄

二阶（加速度）：d²Ω/dt²
  知道"上升在加速还是减速"
  d²Ω/dt² > 0：加速上升（最强信号）
  d²Ω/dt² < 0：减速上升（动能衰减警告）
```

### 3.2 Hawkes 框架对导数体系的扩展

v2.4 引入 Hawkes 后，导数体系从 Ω 域扩展到了 λ 域：

```
零阶：λ(t)
  当前订单到达率
  
一阶：∂λ/∂t
  到达率的变化方向
  
二阶：∂²λ/∂t²
  变化的加速度
  ∂²λ/∂t² < 0 = 衰减在加速 = 离场信号

两个域的对应关系：
Ω 域 → 结构层动态（DOM 配置的演化）
λ 域 → 事件层动态（订单流的自激演化）

策略同时监控两个域的导数：
Ω 域告诉你"结构是否在变化"
λ 域告诉你"动能是否在衰减"
```

## 5. 关系论在 v2.4 中的新表达

### 5.1 鞅作为"关系零点"

```
鞅的定义：E[X_{t+1} | F_t] = X_t
含义：过去和未来之间没有可提取的关系

在关系论中：
鞅态 = 所有关系量为零
  corr(past, future) = 0
  I_Fisher = 0
  n = 0
  
鞅是"关系的缺席"
IS 是"关系的出现"
策略只在关系出现时行动
```

### 5.2 Hawkes 分支比作为"关系的自持性度量"

```
n = α/β

n < 1：关系在衰减（事件间的联系逐渐消失）
n = 1：关系恰好自持（临界点）
n > 1：关系在放大（事件间的联系自我增强）

n 度量的不是"信号有多强"
而是"信号与后续信号之间的关系有多强"
这是信条 6（相位优于幅度）在事件层的体现
```

### 5.3 背景独立性的完整实现

v2.4 的所有度量都不依赖绝对价格：

```
结构层：
  ε → level index（不用价格）
  H_m → 概率分布的熵（无量纲）
  Ω → [0,1] 归一化
  I(Γ) → z-score 标准化

事件层：
  {t_i} → 时间戳（不用价格）
  α, β → 速率参数（单位：1/时间）
  n = α/β → 无量纲比

执行层：
  所有阈值 → 分位数（无量纲）
  所有导数 → 差分/时间（相对变化）

从 Level 0 到决策输出，价格只作为 label 出现
```
